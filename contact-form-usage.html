Contact Form Component Usage Examples

Example 1: Inline Contact Form

Add this to any page where you want an inline contact form:

<!-- Add this to your HTML head -->
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-app-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-firestore-compat.js"></script>
<script src="js/contact-form.js"></script>

<!-- Add this to your HTML body where you want the form -->
<div id="contactFormContainer"></div>

<!-- Add this to your script -->
<script>
document.addEventListener('DOMContentLoaded', function() {
    ContactForm.create(document.getElementById('contactFormContainer'), {
        title: 'Get in Touch',
        showHeader: true
    });
});
</script>

Example 2: Pop-up Contact Form

Add a floating button that opens a pop-up contact form:

<!-- Add this to your HTML head -->
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-app-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-firestore-compat.js"></script>
<script src="js/contact-form.js"></script>

<!-- Add this to your script -->
<script>
document.addEventListener('DOMContentLoaded', function() {
    ContactForm.createPopup({
        position: 'bottom-right'
    });
});
</script>

Example 3: Custom Options

You can customize the form with these options:

ContactForm.create(targetElement, {
    title: 'Contact Us',
    showHeader: true
});

ContactForm.createPopup({
    position: 'bottom-right',
    buttonClass: 'custom-button-class'
});
